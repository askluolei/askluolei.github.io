## 问题

给定 n 个非负整数 a1，a2，...，an，每个数代表坐标中的一个点 (i, ai) 。在坐标内画 n 条垂直线，垂直线 i 的两个端点分别为 (i, ai) 和 (i, 0)。找出其中的两条线，使得它们与 x 轴共同构成的容器可以容纳最多的水。
说明： 你不能倾斜容器，且 n 的值至少为 2。

示例:
```
输入: [1,8,6,2,5,4,8,3,7]
输出: 49
```

## 提交

### 思路1 
暴力解法  
没啥思路可言，直接穷举了

```golang
func maxArea(height []int) int {
	len := len(height)
	result := 0
	for i := 0; i < len; i++ {
		first := height[i]
		for j := i + 1; j < len; j++ {
			second := height[j]
			h := min(first, second)
			w := j - i
			m := h * w
			if m > result {
				result = m
			}
		}
	}
	return result
}

func min(a, b int) int {
	if a < b {
		return a
	}
	return b
}
```

## 思路2
从两侧开始，这样 width 最大，选择一个 height 最大的就行了。然后依次遍历，谁小，就从哪边开始减小下标

```golang
// 2. 从两侧开始，这样 width 最大，选择一个 height 最大的就行了。然后依次遍历，谁小，就从哪边开始减小下标
func maxArea(height []int) int {
	result := 0
	i := 0
	j := len(height) - 1
	for ;i < j; {
		left := height[i]
		right := height[j]

		var m = 0
		if left <= right {
			m = left * (j - i)
			i++
		} else {
			m = right * (j - i)
			j--
		}
		if m > result {
			result = m
		}
	}
	return result
}
```
